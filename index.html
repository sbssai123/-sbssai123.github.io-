P.U.S.H.

Winner of Best Mobile App and Most Innovative Hack Using Bandwidth's API for Calling or Texting at Pearl Hacks 2017

The P.U.S.H. Project, Preventing Unwanted Social Harassment, is an Android App that I developed with two teammates at Pearl Hacks, 2017 UNC Chapel Hill. This mobile app allows the user to quickly notify a select group of friends that they are in danger or in an uncomfortable position with just the push of a button on their home screen. Users are able to get to create multiple groups of friends/ family members and customize messages to be send when in danger. Once a group and message are selected and the app is activated, a notification is stuck to the lock screen of the mobile device for easy access by the user. We used the Bandwidth API which does HTTP POST requests to send a text message with the customized message to the chosen group.

FinesseFitness

Built at HackUMass 2016 and recognized by representatives from Fitbit

Finesse Fitness is an android app that allows user to create workout routines from our predefined exercises or select one of our six preset workouts. Users can create an account on Finesse Fitness and enter their information like name, age, phone number, email, height, and weight. The order and duration of each exercise in the preset and new workouts created are randomized to help spice up the same, boring workout routines. We developed this app using Android Studio and used SQLite to manage and query user, workout, and exercise information.

Music Editor

This is a basic music editor that has the abilities to read tempo and pitches from a text file and play the notes using Java's Midi Sound API. The notes read from the text file were put into our model which sent Midi Messages to Java's Sound API to play the desired notes. The Model-View-Controller design pattern was implemented for scalable and resuable code.
